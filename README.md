## The Claim

Cognition is not exempt from the structural constraints that govern dynamical systems.

If a system admits states, transition dynamics, and constraints, then its evolution is governed by the geometry of its reachable set — not by intention, language, or behavioural compliance.

Cognition is a dynamical process over a state space. What a system can think, perceive, or understand is determined by which states are reachable from its current configuration.

This is not a metaphor. It is a structural condition.

-----

## The Geometry

Every cognitive system — biological or artificial — operates within a state space X governed by transition dynamics:

```
x_{t+1} = F(x_t, u_t)
```

Three objects determine everything:

|Object       |What It Is                                                               |What It Governs                           |
|:------------|:------------------------------------------------------------------------|:-----------------------------------------|
|**ℛ(t)**     |The reachable set — all states accessible from x₀ under admissible inputs|What you *can* think, perceive, understand|
|**Ω**        |The forbidden region — states that must not be reached                   |What must *not* occur                     |
|**S = X \ Ω**|The safe region — everything outside the forbidden set                   |Where the system is allowed to evolve     |

The reachable set is not what the system *does*. It is what the system *can become*.

## Why Cognition Is Not Exempt

If a system has:

- **States** — internal configurations that evolve over time
- **Transitions** — dynamics governing state evolution
- **Constraints** — regions of state space that are inaccessible or forbidden

then it admits a dynamical systems description.

In such systems, behaviour is governed by reachability — the structure of what can be accessed from a given state — not by intention alone.

This implies:

- What a system can think is bounded by its reachable set
- What a system can perceive depends on how inputs deform its state
- What a system can understand depends on whether the target state is reachable from its current configuration

## The Three Illusions

**Illusion 1: “I can think anything.”**

No. A system can only traverse trajectories permitted by its dynamics. States outside ℛ(t) are not difficult to reach — they are structurally inaccessible under current constraints.

**Illusion 2: “Understanding is about intelligence.”**

Not necessarily. Understanding depends on reachability. If a target state T is not reachable from the current state, then it cannot be accessed through reasoning alone.

**Illusion 3: “Safety is behavioural.”**

No. Behaviour reflects trajectories through state space. If unsafe states remain reachable, behavioural compliance does not guarantee safety. Safety requires ℛ(t) ∩ Ω = ∅.

## The Consequences

If cognition is not exempt, then:

1. **Intelligence cannot exceed its reachable set.** No system — human, artificial, biological — can access states outside ℛ(t). Capability is bounded by geometry, not effort.
1. **Safety cannot be patched post-hoc.** If unsafe states are reachable, they will be reached under sufficient exploration. Behavioural correction does not remove reachability.
1. **Understanding cannot be forced.** If the target topology is not reachable from the current state, no amount of information, explanation, or persuasion will produce understanding. The shape must change before the truth can land.
1. **Behaviour is a consequence, not a cause.** Outputs are projections of trajectories through state space. Changing outputs does not change dynamics. Changing dynamics changes everything.

And most importantly:

**You do not fix systems by correcting outputs. You fix them by reshaping the state space.**
